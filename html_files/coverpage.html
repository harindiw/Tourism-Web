Travelling methods you can use in Sri Lanka

Literally three-wheeled open vehicles. Though these can be more expensive than the public busses, it definitely is worth the ride. As the cheapest transportation method, This would be the best option to travel smaller distances as well as longer distance. Trains are the best method for travelling longer distances. Sri Lankan train rides provides amazing views whether it be of the beaches or the mountains. Taxis can be more expensive but traveling in an air conditioned taxi is worth it if you are still getting used to the sunny weather in the island. There are plenty of places where you can quickly rent a cycle if you feel like it. A lot of locals still use cycles for thier transportation as well.
